Project: Bilingual Terminal-Based Banking System
Reliable and accessible software requires solid architectural foundations. In this project, I designed an Object-Oriented Programming (OOP) based banking application in Python that operates directly in the terminal with built-in internationalization (i18n) support.

Instead of standard procedural scripts, I built a custom in-memory data structure to handle real-time account creation, deposits, withdrawals, and secure user-to-user money transfers. The system dynamically switches between English and Turkish interfaces and features a comprehensive transaction history with timestamps for accurate auditing.

This project demonstrates how core programming principles like OOP, robust error handling, and modular design can be applied to create scalable and user-friendly software solutions without relying on external frameworks.
Tech Stack: Python, Object-Oriented Programming (OOP)
Author: Mustafa Alpergün

Proje: İki Dilli Terminal Tabanlı Bankacılık Sistemi
Güvenilir ve erişilebilir yazılımlar, sağlam mimari temeller gerektirir. Bu projede, doğrudan terminal üzerinde çalışan, uluslararasılaştırma (i18n) desteğine sahip ve Nesne Yönelimli Programlama (OOP) mantığıyla tasarlanmış bir bankacılık uygulaması geliştirdim.

Standart prosedürel betikler kullanmak yerine; gerçek zamanlı hesap oluşturma, para yatırma, para çekme ve kullanıcılar arası güvenli para transferi işlemlerini yönetmek için bellekte (in-memory) çalışan özel bir veri yapısı inşa ettim. Sistem, İngilizce ve Türkçe arayüzler arasında dinamik olarak geçiş yapabiliyor ve denetimler için zaman damgalı, kapsamlı bir işlem geçmişi sunuyor.

Bu proje, OOP, hata yönetimi ve modüler tasarım gibi temel programlama prensiplerinin, harici kütüphanelere bağımlı kalmadan ölçeklenebilir ve kullanıcı dostu yazılım çözümleri üretmek için nasıl uygulanabileceğini göstermektedir.
Kullanılan Teknolojiler: Python, Nesne Yönelimli Programlama (OOP)
Yazar: Mustafa Alpergün
